## What This Is

Classical anomaly detection methods ‚ÄĒ z-score, rolling standard deviation, even IsolationForest ‚ÄĒ operate on point statistics and discard the **structural information** encoded in the shape of data.

**TopoSentinel** fixes this.

It extracts topological features from time-series signals using **persistent homology** via Takens delay embedding. Normal signals form clean attractors in phase space; anomalies break that structure. TopoSentinel detects both the anomaly and the moment the underlying data distribution begins to drift ‚ÄĒ before performance degrades.

> **F1 = 0.9782 ¬Ī 0.0140 ¬∑ Recall = 1.000 across all CV folds ¬∑ Drift Precision = 0.9583**

## The Math

### Takens Embedding

Given a scalar time series x(t), the d-dimensional embedding with delay ŌĄ reconstructs the attractor:

```
X(t) = [x(t), x(t+ŌĄ), x(t+2ŌĄ), ..., x(t+(d-1)ŌĄ)]
```

Takens' theorem guarantees this preserves the topology of the original dynamical system. We use **d=3, ŌĄ=31** (selected by hyperparameter search).

### Persistent Homology

As őĶ increases from 0 to ‚ąě, topological features appear (birth b) and disappear (death d):

| Dimension | Symbol | Tracks               |
| --------- | ------ | -------------------- |
| H0        | ő≤‚āÄ     | Connected components |
| H1        | ő≤‚āĀ     | Loops / 1D cycles    |

A normal sin(t) signal produces one dominant H1 feature ‚ÄĒ a single persistent loop. Anomalies break or multiply this structure.

### Boundary Matrices

The boundary operator ‚ąā‚āĖ maps k-chains to (k-1)-chains:

```
‚ąā‚āĀ for triangle {A, B, C}:

     AB   BC   CA
A  [ -1    0   +1 ]
B  [ +1   -1    0 ]
C  [  0   +1   -1 ]

Fundamental lemma: ‚ąā‚āĀ ‚ąė ‚ąā‚āā = 0  (boundary of boundary is empty)

Betti numbers: ő≤‚āĖ = dim(Ker ‚ąā‚āĖ) ‚ąí dim(Im ‚ąā‚āĖ‚āä‚āĀ)
```

### Wasserstein Drift Distance

The drift detector compares the H1 diagram of each incoming window against a reference centroid:

```
W(D‚āĀ, D‚āā) = ő£|pŠĶĘ - q_ŌÉ(i)|   (sorted persistence values, top-50)
```

A window is flagged as drifted when W > calibrated threshold. The threshold is auto-calibrated by sweeping 100 candidates on held-out normal and drifted data, maximising F1.

### Stability Theorem

If two point clouds X and Y satisfy d_H(X, Y) ‚Č§ őī, then:

```
W‚ąě(D(X), D(Y)) ‚Č§ őī
```

This guarantees that small noise perturbations cause only small changes in persistence diagrams ‚ÄĒ the mathematical foundation of TopoSentinel's robustness.

## API Reference

### POST /predict

Classify a 500-sample time-series window.

**Request:**

```json
{
  "signal": [0.012, -0.034, 0.087, ...]   // exactly 500 floats
}
```

**Response:**

```json
{
  "label": 0, // 0 = normal, 1 = anomaly
  "score": -0.470579, // IsolationForest score (higher = more normal)
  "anomaly": false,
  "threshold": -0.605014,
  "latency_ms": 245.6
}
```

**Errors:**

- `422 Unprocessable Entity` ‚ÄĒ signal length != 500
- `500 Internal Server Error` ‚ÄĒ feature extraction failed

## Ablation Studies

### Homology Dimensions

| Features  | Dims    | F1         | Precision  | Recall     |
| --------- | ------- | ---------- | ---------- | ---------- |
| H0 only   | 400     | 0.9878     | 0.9812     | 0.9950     |
| H1 only   | 400     | 0.9426     | 0.9842     | 0.9050     |
| **H0+H1** | **800** | **0.9782** | **0.9577** | **1.0000** |

H0 is the primary discriminator. H1 contributes complementary information ‚ÄĒ their combination achieves perfect Recall.

### Hyperparameter Sensitivity

| Parameter | Tested Values | Locked | Best őĒ             |
| --------- | ------------- | ------ | ------------------ |
| dim       | 2, 3, 4       | **3**  | +0.003 (dim=4)     |
| tau       | 15, 31, 50    | **31** | +0.005 (tau=50)    |
| n_pixels  | 10, 20, 30    | **20** | <0.001 (all equal) |

Locked values are near-optimal. Improvements at higher settings are marginal and not worth the compute cost.

## Thesis Context

This repository is the implementation for a B.Tech Final Year Thesis.

**Thesis title**: _TopoSentinel: Topological Data Analysis for Production-Grade Anomaly Detection_

**Three contributions beyond standard ML projects:**

1. **Topological Wasserstein drift detector** ‚ÄĒ W distance between H1 persistence diagrams as a drift signal. Precision = 0.9583, mathematically principled, not available in any off-the-shelf MLOps tool.

2. **Dual-capability TDA features** ‚ÄĒ a single 800-dimensional representation enables both anomaly classification (Recall = 1.000) and drift detection (Precision = 0.9583). Raw features cannot provide both.

3. **Production-complete pipeline** ‚ÄĒ feature extraction ‚Üí calibrated classifier ‚Üí drift monitor ‚Üí REST API ‚Üí Docker ‚Üí MLflow tracking ‚Üí 97 tests. Most student projects stop at a trained model.
